Die Transformation
==================

Prozesse, die der Datenübertragung dienen, verfügen über eine Transformation.
Die Transformation definiert eine Liste von einzelnen Aktionen, die die Quelldaten verändern oder erweitern.
Das Ziel der Transformation ist die Anpassung des Inhalt an die Anforderung des Ziels.
Die Transformation kann auch funktional genutzt werden.
Zum Beispiel kann die Transformation durch Abfragen Daten generieren, die für Filter oder Suchen genutzt werden können.

Die Transformation setzt sich aus verschiedenen Aktionen zusammen, die in einer festgelegten Reihenfolge ausgeführt werden.
