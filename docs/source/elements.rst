Die Syncler Elemente
====================

Syncler enthält 3 grundlegende Elemente für die Realisierung Ihrer Aufgaben.

Die Verbindungen
----------------
Eine Verbindung stellt die Kommunikation mit einem externen System bereit.
Neben klassischen System wie CRM oder ERP kann sich hinter einer Verbindung aber auch der Eingang für Daten, z.B. Push-Nachricchten oder Dateien
und der Ausgang von Daten, z.B. Serienbriefe verbergen.
